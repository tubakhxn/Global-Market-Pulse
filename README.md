# Global Market Pulse

**Global Market Pulse** is a real-time animated dashboard that visualizes the synchronized movement of multiple global asset classes (equities, crypto, commodities, indices) using Python and matplotlib. It loads historical or live data, normalizes prices, and animates their evolution to help users visually track market co-movement and divergence.

## What is this project about?
- Tracks and visualizes how different asset classes move together or diverge
- Uses real market data (with fallback to mock data)
- Animated, neon-themed matplotlib dashboard
- Focuses on normalized price movement for easy comparison
- Modular, clean Python code

## How to Fork & Run
1. **Fork this repository** to your own GitHub account.
2. **Clone your fork**:
   ```bash
   git clone https://github.com/yourusername/Global-Market-Pulse.git
   cd Global-Market-Pulse
   ```
3. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```
4. **Run the project**:
   ```bash
   python main.py
   ```
